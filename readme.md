## Getting Started

- clone the repo


### Prerequisites

- Run: npm install

- Optional: Run the Tests that comes with the Project (See Running the tests) to ensure that everythings is installed correctly.


## Running the tests

Right now the Project comes with three test files.

1.) npm run test-user
2.) npm run test-product
3.) npm run test-
